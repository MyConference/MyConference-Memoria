\section{App para Android}

La app de MyConference, descrita en el capítulo \ref{chap:android}, es la encargada de mostrar todos los datos introducidos a partir de la web.

Para su desarrollo solo hace falta descargarse el proyecto del repositorio alojado en Git: \url{https://github.com/MyConference/MyConference-Android}.

El IDE utilizado para este proyecto ha sido Eclipse. Basta con importar el proyecto al \textit{workspace} y automáticamente lo compilará. Dentro del repositorio ya se incluyen las librerias necesarias para su ejecución, así que no hace falta nada más.

Para testear la app es necesario un teléfono Android con versión 2.3 o superior, ya que el emulador disponible con el SDK no puede ejecutar nada relacionado con Google Maps.
